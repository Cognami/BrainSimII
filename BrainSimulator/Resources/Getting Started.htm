Getting Started with Brain Simulator II

1. BrainSim networks are like documents—you can load them, edit them, save them, etc.  Many display “Notes” which display automatically explaining each network and how to use it.

2. Try these sample networks to explore various features of the Brain Simulator II. You can find sample network files in the File Menu.

·          BasicNeurons—illustrates the simplest neuron models.

·          CameraTest---inputs the camera on your computer to neuron signals.

·          SpeechTest—uses speech-recognition and synthesis.

·          BabyTalk—learns to speak by trial-and-error learning.

·          Sallie—is an end-to-end AGI model using hearing, vision, and knowledge to learn the meanings of a few words.

·          Maze—shows navigation like a 3-year-old.
